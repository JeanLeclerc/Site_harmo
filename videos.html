Comment ajouter tes vidéos ?

Pour ajouter tes propres vidéos YouTube :

  1. Copie l'URL de ta vidéo YouTube (ex: https://www.youtube.com/watch?v=TON_ID_ICI)
  2. Remplace TON_ID_ICI dans le code ci-dessus par l'ID de ta vidéo (la partie après v=)
  3. Duplique un bloc <article class="video-card"> pour chaque vidéo
  4. Modifie le titre et la description

Tu peux aussi me demander de t'aider à les ajouter !
